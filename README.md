# SQL Basics Practice (MySQL)

This repo contains SQL exercises following a beginner SQL slide deck.

## What’s inside
- `schema.sql` — databases + tables + constraints (PK, FK, DEFAULT, AUTO_INCREMENT)
- `data.sql` — sample data inserts
- `queries.sql` — practice queries: WHERE, ORDER BY, IN, LIKE, GROUP BY, HAVING

## How to run
1. Open MySQL CLI:
   `mysql -u root -p`
2. Run:
   `SOURCE schema.sql;`
   `SOURCE data.sql;`
   `SOURCE queries.sql;`
